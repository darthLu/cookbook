# Cookbook App

A personal digital cookbook built with Flutter.

This app allows users to store, organize, and manage recipes from a mobile device. Recipes can be added, edited, deleted, searched, and categorized, with all data stored locally on the device for offline access.

## Features

### Recipe Management

* Add new recipes
* Edit existing recipes
* Delete recipes
* View detailed recipe instructions

### Organization

* Search recipes by name
* Categorize recipes (Soup, Dessert, Dinner, Breakfast, etc.)
* Filter recipes by category

### Local Storage

* Recipes are saved locally using Hive
* Data persists between app launches
* No internet connection required

### User Experience

* Custom splash screen
* Custom app icon
* Clean and simple interface
* Responsive design for Android devices

## Built With

* Flutter
* Dart
* Hive (Local Database)

## Screens

### Home Screen

Browse all recipes, search by name, and filter by category.

### Recipe Details

View ingredients, cooking instructions, descriptions, and cooking times.

### Recipe Editor

Create new recipes or edit existing ones.

## Installation

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Navigate to the project folder:

```bash
cd cookbook
```

3. Install dependencies:

```bash
flutter pub get
```

4. Run the application:

```bash
flutter run
```

## Motivation

This project was created as a hands-on Flutter learning project and has evolved into a practical cookbook application for managing family recipes. The goal is to continue improving the app while learning mobile development, data storage, and eventually cloud-based synchronization.
